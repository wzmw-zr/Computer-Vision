# 一、模型结构设计

1. 多尺度卷积。
2. $1\times1$卷积降维。
3. 池化保留特征。
4. 中间层输出辅助损失。



# 二、训练技巧

1. 尺度扰动：8%-100%。
2. 长宽比$[3/4,4/3]$。
3. 亮度、饱和度和对比度。



# 三、测试技巧

1. Multi-crop测试。
2. 多模型融合。



# 四、关键点与创新点

1. 大量使用$1\times1$卷积，可降低维度，减少计算量，参数是AlexNet的1/12.
2. 多尺度卷积核，实现多尺度特征提取。
3. 辅助损失层，增加梯度回传，增加正则，减轻过拟合。



# 五、意义

1. 开启多尺度卷积时代。
2. 拉开$1\times1$卷积广泛应用序幕。
3. 为GoogLeNet系列开辟道路。



# 六、启发点

1. 尽管池化会导致空间信息的损失，但在定位、检测和人体姿态识别中仍有应用、延伸拓展。
2. 增加模型深度和宽度，可有效提升性能，但是容易过拟合，计算量也大。
3. **为了节省内存消耗，先将分辨率降低**，再堆叠使用Inception module。
4. **最后一个全连接层，是为了更方便地微调，迁移学习**。
5. **网络中间层特征对于分类也具有判别性。**
6. 学习率下降策略为每8个epoch下降4%，loss曲线很平滑。
7. 数据增强的指导方针：
   + 尺寸在8%-100%。
   + 长宽比在$[3/4,4/3]$。
   + 光照畸变有效。
8. 随机采用差值方法可以提升性能。
9. 10 crop就够了，没必要裁剪144个出来。
